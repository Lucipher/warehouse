## Introduce
The WAREHOUSE is a stroage managment system based on Rails. It is Easy to use, Security and High performance


## Getting start
1. Get the latest code of warehouse
    
    ```bash
    $ git clone git@github.com:shawnyu/warehouse.git
    ```
2. Bundle & Migrate

    ```bash
    $ cd warehouse
    $ bundle install
    $ rake db:create
    $ rake db:migrate
    ```

3. Run the server
    
    ```bash
    $ rails server
    ```
